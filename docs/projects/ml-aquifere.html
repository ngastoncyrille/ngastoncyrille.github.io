Contexte

La cartographie du potentiel aquifère en zone de socle est une problématique majeure en Afrique subsaharienne, où l’accès à l’eau souterraine représente un enjeu vital pour des millions de personnes. Les méthodes classiques reposant sur des approches géophysiques sont coûteuses et limitées à des zones précises.

Ce projet, mené dans le cadre de deux missions consultatives — le Programme d’Approvisionnement en Eau et Assainissement (PAEA) pour le Groupe SGI et le projet ESEPV-Sahel pour le bureau d’études Bugego — a introduit pour la première fois l’intelligence artificielle pour ce type d’analyses au Burkina Faso.

Données et variables utilisées

Les variables d’entrée pour les modèles ML ont été extraites et calculées à partir de diverses sources :

Variable Source Type
Lithologie Carte géologique Catégoriel
Densité de fractures Interprétation Landsat Numérique
Pente (%) MNT SRTM 30m Numérique
Densité de drainage MNT SRTM Numérique
Indice topographique humidité MNT SRTM Numérique
NDVI Landsat 8 Numérique
Distance aux failles Carte tectonique Numérique
Précipitations annuelles Stations météo Numérique
Type de sol Carte pédologique Catégoriel

Méthodologie

Préparation des données

Code 
library(tidyverse)
library(caret)
library(randomForest)
library(rpart)
library(sf)
library(terra)

# Charger les données d'entrainement
# (points forages + variables environnementales)
data_forages <- read.csv("data/forages_variables.csv")

# Variable cible : productivité du forage (Bon/Moyen/Faible)
data_forages$productivite <- as.factor(data_forages$productivite)

# Partition train/test
set.seed(123)
idx_train <- createDataPartition(data_forages$productivite, p = 0.75, list = FALSE)
train_data <- data_forages[ idx_train, ]
test_data  <- data_forages[-idx_train, ]

cat("Données d'entraînement :", nrow(train_data), "observations\n")
cat("Données de test :", nrow(test_data), "observations\n")

Modèle Random Forest

Code 
# Entraînement du Random Forest
set.seed(42)
rf_model <- randomForest(
  productivite ~ lithologie + densite_fractures + pente + 
                 densite_drainage + twi + ndvi + 
                 dist_failles + precipitation + type_sol,
  data       = train_data,
  ntree      = 500,
  importance = TRUE,
  proximity  = TRUE
)

print(rf_model)

Résultats

Performance des modèles

Modèle Précision globale Kappa
Random Forest 84.2% 0.76
Arbre de décision 76.8% 0.65
Régression logistique 71.3% 0.57

Le Random Forest obtient les meilleures performances, avec une précision globale de 84.2% sur le jeu de test.

Variables les plus importantes

L’analyse de l’importance des variables révèle que la densité de fractures et la lithologie sont les facteurs prédictifs les plus influents, suivis de la distance aux failles et du TWI (Topographic Wetness Index).

## Contexte

## Résultats

### Performance des modèles

| Modèle | Précision globale | Kappa |
|--------|------------------|-------|
| Random Forest | **84.2%** | **0.76** |
| Arbre de décision | 76.8% | 0.65 |
| Régression logistique | 71.3% | 0.57 |

Le **Random Forest** obtient les meilleures performances, avec une précision globale de 84.2% sur le jeu de test.

### Variables les plus importantes

L'analyse de l'importance des variables révèle que la **densité de fractures** et la **lithologie** sont les facteurs prédictifs les plus influents, suivis de la **distance aux failles** et du **TWI** (Topographic Wetness Index).
